# Figures

This directory includes proxy-alignment figures whose corresponding scan CSV artifacts are included in `results/exp_proxy/`.

Included figures:

- `proxy_alignment_heart_disease.png` with `results/exp_proxy/heart_disease_scan.csv`
- `proxy_alignment_credit_approval.png` with `results/exp_proxy/credit_approval_scan.csv`
- `proxy_alignment_bank_marketing.png` with `results/exp_proxy/bank_marketing_scan.csv`

Other exploratory proxy-alignment figures from development are not included in this final evidence package unless their underlying scan CSV files are also supplied.
